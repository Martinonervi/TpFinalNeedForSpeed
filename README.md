### Cómo correr
Para compilar y ejecutar el proyecto, ejecutar los siguientes comandos:

```bash
mkdir cmake-build-debug
cd cmake-build-debug
cmake ..
make
```

Luego, se podrá ejecutar el binario generado dentro de `cmake-build-debug`.
